# 📊 Optometry Sales Analysis

## 🧠 Project Overview

This project analyzes optometry sales and customer purchasing behavior using SQL and Power BI.

The goal was to understand:

- Store sales performance
- Customer purchasing trends
- Lens preferences
- Payment method behavior

---

## 🛠️ Tools Used

- MySQL
- SQL Workbench
- Power BI
- GitHub

---

## 🗂️ Dataset Description

| Column | Description |
|---|---|
| Store | Store location/name |
| Price | Product price |
| SaleDate | Date of sale |
| Age | Customer age |
| Gender | Customer gender |
| LensType | Type of lens purchased |
| Payment | Payment method used |

---

## 🧮 SQL Analysis Performed

- Created database and tables
- Inserted and explored sales data
- Analyzed customer behavior
- Identified top-performing stores
- Examined payment trends
- Investigated lens popularity

---

## 📊 Dashboard Insights

The Power BI dashboard helped visualize:

- Sales performance by store
- Popular lens types
- Customer demographics
- Payment method distribution
- Revenue trends

---

## 📈 Key Insights

- Certain stores generated significantly higher sales
- Lens preferences differed across customers
- Card payments were commonly used
- Customer demographics influenced purchasing behavior

---

## 🏁 Conclusion

This project demonstrates how SQL and Power BI can be combined to analyze retail healthcare data and generate business insights for decision-making.
